# MQTT4Snap!

MQTT4Snap! is a Snap! library for using MQTT in Snap! and Snap4Arduino.

The default broker is  [test.mosquitto.org](http://www.mosquitto.org).

### Blocks and usage

Import  mqtt4snap_standalone.xml into Snap!
